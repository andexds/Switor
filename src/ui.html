How it works. At the first create local styles and use them in your design project
Click on the Create theme palette to create visualization of styles
Make a copy of your palette and create new theme, for example dark theme
Select all new colors and click on the Change Colors to change theme
All pages will change theme from your current style to new selected colors

Change theme

Select all new theme color using CTRL (or CMD for mac) and click "Change Colors" for magic

Open file with color styles witch you use like design system.
Add your theme to Switor. Just click Add new theme. Do it with each files with styles
After, when you want to change theme, select frame or layers and click Apply.
If you have a lots of layers Switor may spends more time to change theme.
